papiro azul

Papiro azul web es un sitio estatico báscico con seciones linkeables y formularios de contacto. Fue mi primer proyecto de desarrollo web. En él fueron utilizados html, css, sass, bootstrap y GRID. Este es un tipo de proyecto ideal en el cual solo es necesario mostrar informacion estatica, es decir es una pagina informativa en la cual el usuario podra conocer tu negocio y ponerse en contacto contigo.

GuitarStore

guitar store es una plantilla basica de un e comerce que trae la informacion desde el microservicio de google FIrestore y la procesa para realizar una simulacion de compra. El proyecto incluye un filtro de categorias, una seccion principal, una seccion de detalle de los productos, una seccion de Carrito de compras y finalmente, una seccion de pago donde se genera una orden con un identificador unico y los datos del comprador. Es un tipo de proyecto ideal en el cual es necesario manejar datos y procesar informacion.
